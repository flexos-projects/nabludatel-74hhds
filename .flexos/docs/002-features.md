# Features

## Tier System
Nabludatel is a single-tier, locally run open-source tool. There are no SaaS plans, subscriptions, or paywalls. The user brings their own hardware and Anthropic API key.

## Feature Areas

### 1. Orchestration & Pipeline
**Village-First Queue Management**
- *Description:* Intelligently prioritizes the download and processing queue, starting with remote rural sections and working up to major cities.
- *Behaviours:* Reads from a pre-seeded SQLite database of all Bulgarian sections. Sorts by `is_rural` flag and population size. Resumes automatically from the last uncompleted section if the script is restarted.

**Ephemeral Storage Lifecycle**
- *Description:* Aggressively manages local disk space to allow running on consumer hardware.
- *Behaviours:* Download 1GB+ `.mp4` → Extract audio track → Immediately delete `.mp4` → Run transcription → Delete audio track → Save JSON transcript to database.

**Resilient Networking**
- *Description:* Handles the inevitable dropouts when downloading thousands of gigabytes from a government server.
- *Behaviours:* Implements exponential backoff for failed downloads. Marks sections as `pending_retry` rather than `failed`.

### 2. AI & Analysis
**Local Whisper Transcription**
- *Description:* Converts Bulgarian audio to timestamped text locally using `faster-whisper`.
- *Behaviours:* Bypasses cloud transcription costs. Generates accurate, timestamped JSON arrays of spoken dialogue.

**Headless Forensic Analysis**
- *Description:* Feeds the transcript to Anthropic's Claude via a headless CLI interface (`claude -p`) using prompt caching.
- *Behaviours:* Claude scans for specific typologies: ballot tampering, intimidation, unauthorized persons, unexplained delays. Returns structured JSON containing severity, exact timestamp, trigger quote, and reasoning.

### 3. Command Dashboard (Triage)
**Live Anomaly Feed**
- *Description:* A real-time stream of flagged violations as they are discovered by the pipeline.
- *Behaviours:* Cards appear automatically. Sorted by severity (Critical, Warning). Each card shows the AI reasoning, the exact transcript snippet, and the polling station metadata.

**Instant Verification Links**
- *Description:* Connects the AI's claim directly to the source evidence.
- *Behaviours:* A button on every anomaly card that opens the official `evideo.bg` URL, appending the exact timestamp (e.g., `#t=1h23m45s`) so the journalist can watch the violation immediately.

**System Telemetry**
- *Description:* Live stats on the health and progress of the pipeline.
- *Behaviours:* Displays sections queued, videos downloaded, transcripts generated, disk space saved, and estimated time to completion.

**Coverage Matrix**
- *Description:* A tabular view of all 12,000 sections.
- *Behaviours:* Allows filtering by status (Pending, Clean, Flagged) to identify geographic clusters of anomalies.

## Explicitly Out of Scope
- **Video Hosting:** The system will NOT host or serve video files. It only links to the official government archive.
- **Automated Reporting to Authorities:** The system will NOT automatically email or submit reports to the Central Election Commission. It is a triage tool for humans to verify first.
- **Multi-User Authentication:** The dashboard is a local HTML file served on `localhost`. There are no user accounts, roles, or permissions.
- **Cloud Database Sync:** All state is kept in a local SQLite file. No syncing to Firebase, Postgres, or other cloud databases.
